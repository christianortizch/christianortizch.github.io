TUTORIAL: Cómo instalar Hugo (Windows)

23/03/2020

En este tutorial aprenderás a instalar Hugo en tu pc utilizando la terminal de Windows

Paso 1: Instalar Hugo desde la terminal

Primero debemos abrir la terminal de windows tecleando/buscando cmd en el Inicio de Windows.

Deberías poder abrir esta terminal:

Luego, dentro de la misma terminal que acabamos de abrir, debemos estar en la ruta del disco duro C:\ y escribir el siguiente código:

choco install hugo -confirm

Si queremos la versión Sass/SCSS de hugo extendida escribimos este código en lugar del anterior (recomendado):

choco install hugo-extended -confirm

Para confirmar, en tu disco local C: debería haberse creado la carpeta “Hugo”

¡Y listo! ya tienes Hugo instalado en tu pc
