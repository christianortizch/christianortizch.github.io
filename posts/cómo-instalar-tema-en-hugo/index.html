TUTORIAL: Cómo instalar un tema en Hugo (Windows)

25/03/2020

En este tutorial aprenderás a crear una website e instalar un tema (plantilla de sitio web) de Hugo utilizando la terminal de Windows

Paso 1: Crear una website en Hugo usando la terminal de windows

Para esto debemos abrir la terminal de windows. Tecleando/Buscando cmd en el Inicio de windows. Deberías poder abrir una terminal como ésta:

Una vez dentro de la terminal, debes partir en la ruta de tu disco local C:

Luego, debes teclear el comando cd hugo para acceder a la carpeta hugo en tu terminal.

Una vez estés en la ruta de la carpeta Hugo dentro de la terminal, teclearemos el comando hugo new site website1. Nótese que la palabra website1 en el código corresponde al nombre de la carpeta que vas a crear para tu website, por lo tanto puede ser el nombre que tú estimes conveniente.

Si lo hiciste bien, la terminal te mostrará el siguiente mensaje:

Para confirmar que todo funcionó, en tu carpeta Hugo (en tu disco local C:) se te debiese haber creado una carpeta con el nombre de la website que acabamos de crear, que en este caso es website1

Paso 2: Busca y descarga un tema que te guste

En el sitio https://themes.gohugo.io/ encontrarás una gran diversidad de temas hechos por colaboradores para que puedas comenzar a hacer tu sitio web desde una plantilla ya creada. Esto es ideal para aprender e iniciarse con Hugo!

Una vez seleccionado un tema que te guste, hazle click, haz click en download y te llevará a un sitio de github.com Esto es un repositorio en donde se almacenan todos los archivos que componen el sitio web que elegiste.

Vamos a hacer click en el botón verde que dice lone or Download que se encuentra a la derecha, y luego le daremos a Download ZIP

Una vez descargado el ZIP en donde nosotros queramos, lo vamos a descomprimir y esta nueva carpeta la renombraremos y borraremos la última parte que dice -master.

Paso 3: Instalar el tema descargado en la carpeta de tu website de Hugo

Esta carpeta que hemos descrompimido y renombrado la copiaremos y pegaremos dentro de la carpeta themes que se encuentra dentro de la carpeta website1 que creamos anteriormente.

Ahora tenemos que tener abierto nuestro tema que en este caso es hugo-coder que acabamos de pegar en la carpeta themes que se encuentra dentro de la carpeta website1. También tenemos que tener abierto en otra ventana la carpeta website1

Vamos a abrir la carpeta hugo-coder que acabamos de pegar, y dentro de ella abriremos la carpeta llamada exampleSite.

Ahora debemos copiar todos los archivos que están dentro de la carpeta exampleSite y pegarlos dentro de la carpeta Website1. Notarás que hay carpetas con el mismo nombre y un archivo llamado config.toml. Debemos REEMPLAZAR lo que ya está dentro de la carpeta website1 por lo que estamos pegando desde la carpeta exampleSite

Y con esto ya deberías tener tu tema instalado en la carpeta de tu website que hemos creado mediante la terminal.

Paso 4: Hacer correr la website en el hosting local.

Ahora vamos a la terminal de windows y empezamos desde la ruta del disco local C:

Escribimos el comando cd hugo para ingresar a la carpeta hugo en la terminal.

Luego escribimos el comando cd website1 para ingresar a la carpeta de la website creada.

Por último una vez estemos en esta ruta escribimos el comando hugo server para correr el servidor.

En la terminal debería mostrarte el siguiente mensaje:

Y por último si copias la dirección subrayada con rojo en la imagen de arriba, y la pegas en tu explorador de internet podrás abrir tu sitio web con el tema que le acabas de instalar!

¡Con esto tu tema ya queda listo para comenzar a editarlo!
